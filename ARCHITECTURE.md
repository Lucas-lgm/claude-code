## 1. 项目概述

Claude Code 是 Anthropic 官方的 AI 编程助手 CLI 工具。它允许用户在终端中与 Claude 进行交互式对话，Claude 可以自主使用工具（读写文件、执行命令、搜索代码、浏览网页等）来完成软件工程任务。

**核心能力：**
- 交互式 REPL 终端界面（React/Ink 渲染）
- 40+ 内置工具的 Agentic 自主循环
- 100+ 斜杠命令
- 文件级持久化记忆系统
- 可扩展的 MCP 协议/插件/技能
- 多 Agent 协调与子 Agent 生成
- IDE 集成（VS Code Bridge）


**关键机制：**
- **Token Budget Tracking** — 每轮追踪 token 使用量，接近限制时触发压缩
- **Context Compaction** — 三种策略：microcompact(微压缩)、snip(截断)、reactive compact(响应式压缩)
- **Tool Result Storage** — 管理工具结果大小，防止上下文爆炸
- **Post-Sampling Hooks** — 响应后置处理（分类器、顾问）
